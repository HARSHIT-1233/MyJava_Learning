# Java SE 21 Developer Certification Preparation

This repository contains my Java practice programs and notes created while preparing for the **Java SE 21 Developer Certification**.

The goal of this repository is to strengthen core Java concepts through hands-on coding and maintain a collection of examples that align with certification objectives.

## Requirements

* Java SE 21 or later
* Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or command line

## Compile and Run

Compile:

```bash
javac FileName.java
```

Run:

```bash
java FileName
```

Example:

```bash
javac HelloWorld.java
java HelloWorld
```
