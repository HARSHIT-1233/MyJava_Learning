# Java Arrays Examples

This repository contains Java programs demonstrating the fundamentals of arrays, multidimensional arrays, searching, and sorting using the `Arrays` utility class.

## Topics Covered

- Array Declaration, Initialization, and Manipulation
- One-Dimensional Arrays
- Multi-Dimensional Arrays (2D & 3D)
- Array Searching using Binary Search
- Array Sorting using Arrays.sort()
- Working with Primitive and Object Arrays

---

## Files Overview

### 1. Array.java

Demonstrates the basics of arrays in Java:

- Array declaration and instantiation
- Array initialization
- Accessing array elements
- Modifying array values
- Finding array length
- Working with different data types inside arrays
- Using `Arrays.toString()` for printing arrays

**Concepts Learned**
- One-dimensional arrays
- Array indexing
- Array length property
- Object arrays

---

### 2. MultiDimensionalArrays.java

Demonstrates:

- Two-Dimensional Arrays (2D Arrays)
- Three-Dimensional Arrays (3D Arrays)
- Jagged Arrays (arrays with different row sizes)
- Accessing nested array elements

**Concepts Learned**
- Matrix-like structures
- Nested arrays
- Multi-level indexing

---

### 3. SearchingInArrays.java

Demonstrates searching techniques using the `Arrays` utility class.

Topics covered:

- Binary Search on sorted arrays
- Behavior of binary search when element is absent
- Binary Search on unsorted arrays
- Searching in String arrays
- Sorting before searching

**Concepts Learned**
- `Arrays.binarySearch()`
- Importance of sorted data for binary search
- Return values for successful and unsuccessful searches

---

### 4. SortingArray.java

Demonstrates sorting various types of arrays using `Arrays.sort()`.

Arrays sorted:

- Double Arrays
- Integer Arrays
- Character Arrays
- String Arrays

**Concepts Learned**
- Ascending order sorting
- Lexicographical sorting
- Unicode-based character sorting
- String sorting behavior in Java

---

## How to Run

### Compile

```bash
javac *.java
```

### Run

```bash
java Array
```

```bash
java MultiDimensionalArrays
```

```bash
java SearchingInArrays
```

```bash
java SortingArray
```

After completing these programs, you will understand:

- How arrays are stored and accessed in Java
