# Java Comments Example

## Description

This is a simple Java program that demonstrates the different types of comments used in Java.

The program includes:

* Single-line comments (`//`)
* Multi-line comments (`/* */`)
* Documentation comments (`/** */`)

It also contains a method that multiplies two integers and prints the result.

## Output

Hello Java Developers

Multiplication = 225

## Features

* Demonstrates Java commenting styles.
* Uses a separate method for multiplication.
* Shows how to write JavaDoc comments for methods.

## How to Run

1. Compile the program:

   ```bash
   javac JavaComments.java
   ```

2. Run the program:

   ```bash
   java JavaComments
   ```
