# Java Decision Making Statements

This project demonstrates the various decision-making constructs available in Java. These statements allow programs to execute different blocks of code based on conditions and user input.

## Topics Covered

- if Statement
- if-else Statement
- Nested if Statement
- if-else-if Ladder
- switch Statement
- Ternary Operator

## Files Included

### 1. IfElseStatement.java
Demonstrates:
- Checking whether a number is even or odd
- Using boolean conditions with if-else
- Multiple independent if statements
- User input using Scanner

### 2. IfLatterNestedIfStatement.java
Demonstrates:
- Finding the largest of three numbers using nested if statements
- Age classification using if-else-if ladder
- Nested conditional logic
- User input handling

Age Categories:
- Child (< 13)
- Teenager (13–18)
- Adult (19–64)
- Senior (65+)

### 3. SwitchCaseStatement.java
Demonstrates:
- Using switch statements with integer values
- Grouping multiple cases
- Determining weekdays and weekends
- Mapping day numbers to day names
- Using the default case

### 4. TernaryOperator.java
Demonstrates:
- Replacing simple if-else statements using the ternary operator
- Conditional value assignment
- Understanding post-increment and post-decrement operations within ternary expressions

## Prerequisites

- Java JDK 21 or later
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code, NetBeans)

## How to Run

### Compile

```bash
javac *.java
```

### Execute

```bash
java IfElseStatement
```

```bash
java IfLatterNestedIfStatement
```

```bash
java SwitchCaseStatement
```

```bash
java TernaryOperator
```

## Learning Outcomes

After completing these examples, you will understand:

- How decision-making works in Java
- When to use if, if-else, and nested if statements
- How if-else-if ladders help manage multiple conditions
- How switch statements improve readability for multiple choices
- How the ternary operator provides concise conditional expressions
- Best practices for handling user input and conditional logic
