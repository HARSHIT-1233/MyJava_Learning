# Java Input and Output Examples

This repository contains basic Java programs demonstrating console input and output operations.

## Files Included

### 1. BasicOutput.java

Demonstrates different methods of displaying output in Java:

* `System.out.println()`
* `System.out.print()`
* `System.out.printf()`

Also includes a simple arithmetic example using formatted output.

#### Concepts Covered

* Console Output
* Formatted Printing
* Variables and Expressions

---

### 2. BasicInput.java

Demonstrates how to read user input from the keyboard using the `Scanner` class.

#### Concepts Covered

* Scanner Class
* Reading Integer Input
* User Interaction
* Resource Management using `close()`

---

## How to Compile

```bash
javac BasicOutput.java
javac BasicInput.java
```

## How to Run

```bash
java BasicOutput
java BasicInput
```
