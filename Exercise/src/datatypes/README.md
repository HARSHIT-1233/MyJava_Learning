# Java Data Types and Type Casting Examples

This repository contains Java programs demonstrating fundamental Java concepts such as primitive data types, character manipulation, and type casting.

## Files Included

### 1. DataTypes.java

Demonstrates Java primitive data types:

* boolean
* char
* byte
* short
* int
* long
* float
* double
* String

Also shows the use of numeric literals with underscores for improved readability.

### 2. TypeCasting.java

Demonstrates:

* Implicit (Widening) Type Casting

  * int → long → double

* Explicit (Narrowing) Type Casting

  * double → long → byte

Helps understand how Java converts values between different data types.

### 3. CharExample.java

Demonstrates:

* Character data type (`char`)
* ASCII/Unicode character representation
* Character utility methods:

  * Character.isLetter()
  * Character.isDigit()
* Character arithmetic and increment/decrement operations

## How to Compile

```bash
javac *.java
```

## How to Run

```bash
java DataTypes
java TypeCasting
java CharExample
```

## Topics Covered

* Java Primitive Data Types
* String Basics
* Character Handling
* ASCII and Unicode Values
* Widening Type Casting
* Narrowing Type Casting
* Character Utility Methods
