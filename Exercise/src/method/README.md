# Java Methods

## 📌 Overview

This project demonstrates the fundamentals of **Methods in Java**. It covers how to create methods, pass parameters, return values, overload methods, and use variable-length arguments (VarArgs). These examples help beginners understand how methods improve code reusability, readability, and modularity.

---

## 📂 Files Included

| File Name                | Description                                                                                                    |
| ------------------------ | -------------------------------------------------------------------------------------------------------------- |
| `Methods.java`           | Demonstrates different method declarations, access modifiers, return types, and parameters.                    |
| `MethodExample.java`     | Shows how to create, call, and return values from methods.                                                     |
| `MethodOverloading.java` | Demonstrates method overloading by defining multiple methods with the same name but different parameter lists. |
| `VarArgsExample.java`    | Demonstrates Variable Arguments (VarArgs) using `String...` to accept multiple inputs.                         |

## ▶️ How to Run

1. Save all Java files inside the `method` package.
2. Open a terminal or command prompt.
3. Compile all files:

```bash
javac method/*.java
```

4. Run any program individually.

Example:

```bash
java method.MethodExample
```

or

```bash
java method.MethodOverloading
```

or

```bash
java method.VarArgsExample
```

## 📖 Program Descriptions

### 1. Methods.java

Demonstrates various method declarations, including:

* Methods without return values
* Methods returning values
* Different access modifiers
* Methods with parameters
* Static and non-static methods

---

### 2. MethodExample.java

Illustrates:

* Calling methods
* Passing arguments
* Returning values
* Calculating the sum of two numbers
* Finding the minimum of two numbers

Sample Output:

```text
Please enter two numbers:
10
25
You entered 10 and 25
Sum of numbers: 35
Minimum number is: 10
```

---

### 3. MethodOverloading.java

Shows how multiple methods can share the same name but have different parameter lists.

Example Output:

```text
Sum of two numbers: 9
Sum of three numbers: 15
Sum of two double numbers: 9.46
```

---

### 4. VarArgsExample.java

Demonstrates Java VarArgs (`...`) by allowing the user to enter any number of integers, which are then summed.

Example:

```text
Please enter the numbers you want to sum, leave a space between them

10 20 30 40

10 + 20 + 30 + 40 = 100
```
